Global Changes

TISEAN_2.1 -> TISEAN 3.0.x

  • Starting from this release TISEAN is published under the terms of the GPL (version 2 or later). We hope that this helps all people who want to implement TISEAN into other open source packages.
  • We extended the version number. The new last number will be incremented in case we just fix bugs, while the central number will be incremented if new features are added.
  • We removed the mailing list and the mailing account. In case you miss one of these two features send your complaints to the guys who think that the world needs spam mails.

TISEAN_2.0 -> TISEAN_2.1

  • There are now suggested exercises which also come with some experimental data to play with.
  • The paper version of the manual for surrogate data has now appeared as [Thomas Schreiber and Andreas Schmitz, Surrogate time series, Physica D 142 346 (2000)]
  • Further redundancies between the former Wuppertal/Dresden branches and between multivariate/univariate programs have been removed. Again, the sources of the removed programs are still in the unsupported directory.

TISEAN_1.2 -> TISEAN_2.0

  • As you may haven noticed, the layout of the manuals has changed quite a bit. In particular, the distinction between a Dresden and a Wuppertal branch is no longer made. Even internally it is softening up.
  • There are no more C++ programs in the package - one compiler less needed!
  • There are no more C programs on the Wuppertal side, but that matters only for Thomas.
  • The internal distinction Wuppertal/Dresden is now replaced by the distinction C/FORTRAN.
  • A number of redundancies between the former Wuppertal/Dresden branches has been removed. We hope this makes life easier for many users. We apologize to those who are missing their choice. The sources of the removed programs are still in the unsupported directory.
  • We adopted a new concept for the Windows executables.

TISEAN_0.9 -> TISEAN_1.2

  • The paper
    R. Hegger, H. Kantz, and T. Schreiber,
    Practical implementation of nonlinear time series methods: The TISEAN package
    CHAOS 9, 413 (1999)
    has been included in html format as an extended documentation. Paper versions are also available but not part of this distribution.

TISEAN_0.5 -> TISEAN_0.9

  • Simplified installation through a configure script.
  • Quite a number of options have changed their names in order to re-unify (re-?) them between the Dresden and Wuppertal branches.
  • Everything not an option or argument to an option is now treated as a file name. Thus you may place input file names wherever you want.
  • If no file name is given, usually "-" is assumed, that is, input is read from stdin.
  • Experimental DOS executables available.
  • Centralised installation notes etc.
  • Major but for the user irrelevant changes in the structure of the source tree and Makefile's.
